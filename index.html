아래 항목별로 현재 샘플 데이터가 어떻게 생겼는지, 실제 데이터로 어떻게 교체하는지 설명합니다.
🚢 항로 운임 데이터 (SCFI)
현재 샘플 형태
{id:'busan-rotterdam', scfi:1842, change:5.2, vol:98, risk:22, profit:91}
실제 데이터 출처
상하이항운교역소 (SCFI) — 매주 금요일 발표 Baltic Exchange (FBX) — 실시간 API 제공 한국해양수산개발원 KMI — 국내 물동량
교체 방법
ROUTES_DATA 객체의 scfi, change 값을 API 응답값으로 교체. change는 전주 대비 % 변동률.
⚠️ 위험구간 데이터 (riskZones)
현재 샘플 형태
{name:'홍해 구간', lat:15, lng:42, riskLevel:75, desc:'후티 위협 잔재'}
실제 데이터 출처
IMB 해적 위험 리포트 (국제해사국) BIMCO 항행 경보 — 이메일 구독 가능 미 해군 NAVCEN — 항행경보 공개 API
riskLevel은 0~100 수치. 61+ 빨강, 31~60 노랑, 0~30 초록으로 자동 표시됨.
💰 ROI 계산 변수
현재 샘플 기본값
TEU: 300 | 비용/TEU: $800 | 적재율: 85% | 항차: 4회/월
실제 적용 방법
선사 내부 운항원가 시스템(ERP)에서 항로별 실제 비용 데이터 추출. 적재율은 BL 발행 데이터 기반. 항차는 서비스 루프 스케줄 기반으로 설정.
내부 ERP 연동 (SAP / Oracle) 선사 스케줄 DB (항차·기항지)
📰 실시간 뉴스
현재 샘플 형태
{headline:'...', source:'Reuters', time:'2시간 전', impact:'pos', sentiment:'pos'}
실제 데이터 출처
⚠️ Reuters Connect API — 유료 계약 필요 ⚠️ Bloomberg Terminal Feed — 유료 Google News RSS — 무료, 정확도 낮음 TradeWinds / Lloyd's List RSS — 해운 전문
sentiment(pos/neg/neu)와 impact는 Claude API로 뉴스 본문 분석 후 자동 분류 가능.
👥 사용자 & 조직 데이터
현재 샘플 (하드코딩)
users: [{name:'김선복', role:'선복팀장', color:'#1D9E75'}, ...]
실제 적용 방법
사내 Active Directory / HR 시스템 연동으로 사용자 자동 로드. 역할별 권한(어드민/팀장/담당자/조회전용) 구분 필요.
SSO (Single Sign-On) 연동 사내 LDAP / Active Directory
⚙️ 실제 구현 로드맵
Claude Code로 구현 가능한 단계별 연동 가이드입니다.
1
🗄️ 백엔드 API 서버 구성
Node.js(Express) 또는 Python(FastAPI)으로 REST API 서버 구축. SCFI 데이터 크롤러(매주 금요일 자동 실행) + 뉴스 RSS 파서 스케줄러 구성.
Node.js / Python · PostgreSQL · cron
2
🔌 외부 API 연동
Baltic Exchange FBX API 키 발급 → /api/freight 엔드포인트 생성. 뉴스는 Google News RSS 무료 파싱부터 시작해 Claude API로 감성 분석 자동화.
FBX API · Google RSS · Claude API
3
📧 이메일 긴급공지 발송
SendGrid 또는 AWS SES 연동. 긴급공지 발령 시 수신자 그룹(전체/팀별) 자동 이메일 발송. HTML 템플릿으로 위험등급별 색상 차별화.
SendGrid API · AWS SES · Nodemailer
4
💾 실시간 협업 DB
어드민 의견·데일리체크·회의록을 DB에 영속화. 현재는 새로고침 시 초기화됨 — Firebase Realtime DB 또는 Supabase로 실시간 동기화 구현.
Firebase · Supabase · WebSocket
5
🔐 사용자 인증 & 권한
사내 SSO 연동 또는 자체 JWT 인증. 역할별 권한(어드민/팀장/담당자/조회전용) 구분. 공지 발령은 팀장 이상만 가능하도록 제한.
JWT · OAuth2 · SSO
💡 Claude Code 활용 팁: 각 단계를 Claude Code에게 "Node.js로 SCFI 데이터 크롤러 만들어줘" 형태로 요청하면 실제 코드를 바로 생성해줍니다. 노코드 베이스로 시작해서 점진적으로 확장 가능합니다.
